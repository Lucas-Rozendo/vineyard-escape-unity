# Vineyard Defender

## Resumo do projeto

**Vineyard Defender** é um jogo 2D desenvolvido na Unity, com temática de fantasia medieval baseada em vinhedos, uvas e armas. O jogador controla um aventureiro que precisa defender o vinhedo contra ondas de inimigos goblins, utilizando arco, espada e uma habilidade especial carregada por uvas coletáveis no mapa.

O jogo possui movimentação top-down, sistema de combate, waves progressivas, inimigos com perseguição, boss final, HUD, menu principal, tela de pausa, tela de créditos, vitória e derrota.

## Projeto acadêmico

Este jogo foi produzido durante as aulas da disciplina de desenvolvimento de jogos, com o objetivo de aplicar conceitos de programação em C#, criação de mecânicas em Unity, organização de cenas, controle de personagens, inimigos, interface, áudio e build final para Windows.

## Como executar o jogo

O repositório contém uma pasta `Build` com a versão compilada para Windows.

Para jogar:

1. Abra a pasta `Build`.
2. Execute o arquivo `.exe` do jogo.
3. Mantenha todos os arquivos da pasta `Build` juntos, pois o executável depende das pastas e arquivos gerados pela Unity.

## Como abrir o projeto na Unity

1. Clone ou baixe este repositório.
2. Abra o Unity Hub.
3. Clique em `Add project from disk`.
4. Selecione a pasta do projeto.
5. Abra o projeto na versão da Unity utilizada no desenvolvimento.
6. Abra a cena do menu ou a cena principal do jogo.
